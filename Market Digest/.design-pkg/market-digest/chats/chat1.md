Redesigned as a clean, Apple-style interactive prototype with a companion design system.

**Market Digest.html** — single scrolling page with sticky frosted nav and a segmented control that scroll-spies between four sections:

- **Overview** — Hero NIFTY 50 with 84px display numerics, sparkline, 4-stat row with inline range bars, candlestick chart with 1M/3M/All toggle, and twin insight cards (Today's Read + Key Levels).
- **Flows** — Four KPI cards, FII vs DII grouped bar chart, hairline-row table.
- **Volatility** — VIX hero card with chips + sparkline, twin "near 52W high/low" tables.
- **Global** — Indices grid, currency cards, and a featured dark-glass GIFT Nifty card.

**Design System.html** — principles, the full color set (neutrals + semantic), the type scale shown live, spacing/radius tokens, and every component (segmented control, chips, deltas, buttons, stat cells, range bars, level rows, tables, icons, chart legend).

Aesthetic notes — SF Pro display stack with JetBrains Mono for tokens, tabular numerics everywhere, hairline dividers instead of heavy borders, color used only as signal (green/red for direction), and a single warm accent for the EMA line. Charts restyled to grayscale-by-default so up/down can carry semantic meaning.

Footer of the prototype links to the design system page.
